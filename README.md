# Video Processing Dashboard

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Real-time: Socket.IO
- Uploads: Multer
- Charts: Recharts

## Features
- Video upload (MP4)
- Real-time processing progress
- Status tracking
- Analytics dashboard
- Video streaming
- Admin / Viewer roles (UI)

## How to Run
### Backend
cd backend
node index.js

### Frontend
cd frontend
npm run dev
